vue项目中如何使用less

1、首选，安装 less 和 less-loader npm install less less-loader –save-dev
2、在build/webpack.base.conf.js文件中的loader/rules中添加配置

3、最后，在代码中的 style 标签中 加上 lang=”less” 属性即可
